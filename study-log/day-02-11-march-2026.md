## Key Concepts Learned

### TCP/IP Model
The TCP/IP model explains how data moves across networks using layered communication.

Layers studied:

1. Application Layer
2. Transport Layer
3. Network Layer
4. Data Link Layer
5. Physical Layer

Encapsulation process:

Data → Segment → Packet → Frame → Bits

---

### Ethernet Frame

Ethernet frames are used to send data within a local network (LAN).

Frame structure includes:

- Preamble
- Destination MAC Address
- Source MAC Address
- Type Field
- Data (Payload)
- Frame Check Sequence (FCS)

Switches use the **Destination MAC Address** inside the frame to determine where to forward the data.
